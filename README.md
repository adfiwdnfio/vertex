# Vertex

Live intelligence dashboard for markets, crypto, AI, US politics, and prediction-market signals.

The site is static and deploys on GitHub Pages. It does not require private API keys:

- Crypto prices: CoinGecko
- Fear & Greed: alternative.me
- News: Google News RSS through rss2json
- Prediction markets: Polymarket Gamma API
- Market snapshot: Stooq and CoinGecko via the scheduled GitHub Actions updater

Deployment target: https://adfiwdnfio.github.io/vertex/
